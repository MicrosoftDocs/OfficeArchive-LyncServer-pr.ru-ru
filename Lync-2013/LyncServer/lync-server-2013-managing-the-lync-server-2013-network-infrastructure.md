# <a name="managing-the-lync-server-2013-network-infrastructure"></a>Управление инфраструктурой сети Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Тема последнего изменения:** 2014-02-11_

Microsoft Lync Server 2013 включает поддержку управления допуском звонков (CAC) и обход мультимедиа. Для реализации этих функций необходимо настроить сеть регионов, сайтов, подсетей и т. д., благодаря которой вы сможете управлять пропускной способностью в ситуациях, когда необходимо ограничить передачу звука и видео. Кроме того, вы можете использовать сетевую технологию качества обслуживания (QoS) для обеспечения оптимального взаимодействия с конечными пользователями для голосовой и видеосвязи.

С помощью панели управления Lync Server вы можете настроить CAC, обход мультимедиа и качество обслуживания, а также управлять ими. В следующих разделах приведены инструкции по выполнению этой задачи.
