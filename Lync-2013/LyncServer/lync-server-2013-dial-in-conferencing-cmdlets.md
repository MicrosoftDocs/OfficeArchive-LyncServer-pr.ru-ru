_**Тема последнего изменения:** 2012-03-21_

Конференц-связь с телефонным подключением — это способ присоединиться к звуковой части конференции с помощью обычного телефонного телефона (то есть устройства в телефонной сети, подключенного к общественной коммутируемой связи).

<div>

## <a name="dial-in-conferencing-cmdlets"></a>Командлеты конференц-связи с телефонным подключением

Если вы не хотите разрешать Конференц-связь с телефонным подключением, вы можете отключить эту возможность с помощью командлета Set-CsConferencingPolicy и задать для свойства EnableDialInConferencing значение false. По умолчанию всем пользователям разрешено размещать собрания, которые включают конференц-связь с телефонным подключением.
