# <a name="planning-for-devices-in-lync-server-2013"></a>Планирование устройств в Lync Server 2013

_**Тема последнего изменения:** 2013-02-28_

Lync Server 2013 включает Lync Phone Edition, программное обеспечение, которое работает на соответствующих устройствах и предоставляет традиционные и расширенные возможности телефонии, встроенную безопасность, управляемость и многое другое. Lync Phone Edition работает аналогично с Lync Server 2013, как это происходит в Lync Server 2010. Сведения о планировании устройств можно найти в разделе [планирование устройств](https://go.microsoft.com/fwlink/p/?linkid=285880) в библиотеке TechNet Server 2010.
