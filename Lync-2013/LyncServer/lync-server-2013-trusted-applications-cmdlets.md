# <a name="trusted-applications-cmdlets-in-lync-server-2013"></a>Командлеты доверенных приложений в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

</div>

_**Тема последнего изменения:** 2012-06-20_

Надежное приложение — это приложение, разработанное третьим лицом, которое получает состояние Trusted для работы в составе Microsoft Lync Server 2013, но не является встроенной частью продукта. Lync Server 2013 предоставляет командлеты, которые можно использовать для настройки доверенных приложений и управления ими.

<div>
