# <a name="managing-enhanced-9-1-1-and-the-location-service-in-lync-server-2013"></a>Управление службой Enhanced 9-1-1 и службой расположения в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Тема последнего изменения:** 2012-11-01_

Lync Server 2013 поддерживает вызов улучшенных 9-1-1 (E9-1-1) на клиентах Lync и устройствах Lync Phone Edition. Если вы настроили Lync Server 2013 для E9-1-1, экстренные вызовы, размещенные в Lync 2013 или Lync Phone Edition, включают в себя сведения о местоположении ответа на аварийную информацию (ERL) из базы данных службы сведений о расположении. С помощью описанных в этом разделе процедур можно управлять политикой расположения.
