# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>Необязательно Проверка развертывания объявления в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**Тема последнего изменения:** 2013-02-25_

После установки и настройки приложения "Оповещение" необходимо проверить конфигурацию, чтобы убедиться, что вызовы на неназначенные номера работают соответствующим образом. Как минимум, необходимо проверить следующее:

  - вызвать номер, который действителен в организации, но никуда не назначен;

  - вызвать неназначенный номер и проверить, воспроизводится ли правильное оповещение.
