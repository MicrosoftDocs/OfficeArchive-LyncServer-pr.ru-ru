# <a name="deploying-devices-in-lync-server-2013"></a>Развертывание устройств в Lync Server 2013

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**Тема последнего изменения:** 2013-09-10_

Lync Server 2013 включает Lync Phone Edition, программное обеспечение, которое работает на соответствующих устройствах и предоставляет традиционные и расширенные возможности телефонии, встроенную безопасность, управляемость и многое другое. Lync Phone Edition работает аналогично с Lync Server 2013, как это происходит в Lync Server 2010. Подробнее о том, как развертывать устройства, можно найти в разделе [развертывание Lync Phone Edition](https://go.microsoft.com/fwlink/p/?linkid=285880) в библиотеке TechNet Server 2010.
